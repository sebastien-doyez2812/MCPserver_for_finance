## Run Locally

Clone the project

```bash
  git clone https://github.com/sebastien-doyez2812/MCPserver_for_finance.git
```

Go to the project directory

```bash
  cd MCPserver_for_finance
```

Install dependencies

```bash
pip install -r requirements.txt
```

Start the server
```bash
uv run mcp dev server.py

```
Run the agent.py
```bash
python agent.py

```
